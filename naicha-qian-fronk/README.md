# 奶茶销售系统前台

一个基于Vue 3的现代化奶茶销售系统前台界面，提供完整的用户购物体验。

## 功能特性

### 🏠 首页
- 轮播图展示
- 热门商品推荐
- 商品分类导航
- 最新活动展示

### 👤 账号管理
- 用户注册/登录
- 个人信息管理
- 安全设置（密码修改、手机验证等）
- 用户状态管理

### 🛍️ 商品浏览与查询
- 商品列表展示
- 多维度筛选（分类、价格、排序）
- 商品搜索功能
- 商品详情查看
- 收藏功能

### 🛒 购物车管理
- 购物车商品管理
- 数量调整
- 商品选择
- 优惠券应用
- 推荐商品

### 📦 订单管理
- 订单列表查看
- 订单状态跟踪
- 订单操作（付款、取消、再次购买）
- 订单统计

### 🎉 营销活动
- 活动列表展示
- 优惠券管理
- 活动参与
- 会员专享

## 技术栈

- **Vue 3** - 渐进式JavaScript框架
- **Vue Router** - 官方路由管理器
- **Pinia** - 状态管理
- **Element Plus** - UI组件库
- **Vite** - 构建工具
- **Axios** - HTTP客户端

## 项目结构

```
src/
├── components/          # 公共组件
├── layout/            # 布局组件
│   └── index.vue      # 主布局
├── router/            # 路由配置
│   └── index.js
├── stores/            # 状态管理
│   ├── user.js       # 用户状态
│   └── cart.js       # 购物车状态
├── views/            # 页面组件
│   ├── Home.vue      # 首页
│   ├── Login.vue     # 登录页
│   ├── Register.vue  # 注册页
│   ├── Profile.vue   # 个人中心
│   ├── Products.vue  # 商品列表
│   ├── Cart.vue      # 购物车
│   ├── Orders.vue    # 订单管理
│   └── Activities.vue # 营销活动
├── App.vue           # 根组件
└── main.js           # 入口文件
```

## 安装和运行

### 环境要求
- Node.js >= 16.0.0
- npm >= 8.0.0

### 安装依赖
```bash
npm install
```

### 开发环境运行
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
```

### 预览生产版本
```bash
npm run preview
```
