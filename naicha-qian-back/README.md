# 奶茶商城后台管理系统

## 项目简介

奶茶商城后台管理系统是一个基于 Vue 3 + Element Plus 的现代化后台管理平台，为奶茶商城提供完整的管理功能。

## 技术栈

- **前端框架**: Vue 3
- **UI组件库**: Element Plus
- **状态管理**: Pinia
- **路由管理**: Vue Router 4
- **构建工具**: Vite
- **样式预处理**: Sass

- 平台数据概览

- 订单接收与处理

## 安装和运行

### 环境要求
- Node.js >= 16.0.0
- npm >= 8.0.0

### 安装依赖
```bash
npm install
```

### 开发环境运行
```bash
npm run dev
```

### 生产环境构建
```bash
npm run build
```

### 预览构建结果
```bash
npm run preview
```

## 开发说明

### 登录测试
项目提供了模拟登录功能：
- 管理员登录：任意用户名密码
- 商家登录：任意用户名密码

### 页面开发
- 所有页面都基于 Element Plus 组件库
- 使用 Vue 3 Composition API 编写
- 支持 TypeScript（可选）

### 样式规范
- 使用 SCSS 预处理器
- 遵循 BEM 命名规范
- 响应式设计原则
